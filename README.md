# Веб-система визуализации нарушений на карте Москвы

Проект предназначен для отображения нарушений на интерактивной карте Москвы и загрузки актуальной XLSX-выгрузки через административный раздел.

## Возможности

- отображение карты Москвы;
- фильтрация данных по дате, статусу, категории, группе объекта и объекту;
- просмотр карточки нарушения по клику на точку;
- вход администратора;
- импорт XLSX-файла;
- перезапись актуального набора данных при новой загрузке;
- хранение данных в SQLite.

## Запуск на macOS или Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

Адрес приложения:

```text
http://127.0.0.1:8000
```

## Запуск на Windows

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

Адрес приложения:

```text
http://127.0.0.1:8000
```

## Вход администратора

```text
admin
admin123
```

## Импорт данных

Для загрузки данных нужно открыть карту, перейти в раздел администратора, выполнить вход и выбрать XLSX-файл.
Обязательные столбцы:

```text
ID задачи
Широта
Долгота
Время назначения задачи
```

Дополнительные столбцы:

```text
Категория задачи
Группа объекта
Название объекта
Статус задачи
Ссылка на задачу
Нарушение
Устранение
```
